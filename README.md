# API REST – Rent-a-Car

Sistema de aluguer de viaturas desenvolvido em Node.js.

## Funcionalidades

- Gestão de carros
- Gestão de clientes
- Gestão de reservas

## Rotas

POST /api/carros
GET /api/carros
PUT /api/carros/:id
DELETE /api/carros/:id

POST /api/clientes
GET /api/clientes

POST /api/reservas
GET /api/reservas

## Tecnologias

Node.js, Express, MongoDB, Jest

## Execução

npm install
npm run dev

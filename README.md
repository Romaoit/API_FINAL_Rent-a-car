# API REST – Rent-a-Car

Sistema de aluguer de viaturas desenvolvido em Node.js.

## Funcionalidades

- Gestão de carros
- Gestão de clientes
- Gestão de reservas

## Rotas

POST /api/cars
GET /api/cars
PUT /api/cars/:id
DELETE /api/cars/:id

POST /api/clients
GET /api/clients

POST /api/reservations
GET /api/reservations

## Tecnologias

Node.js, Express, MongoDB, Jest

## Execução

npm install
npm run dev
